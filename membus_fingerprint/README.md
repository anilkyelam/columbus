# CSE227
#Project - Membus Fingerprinting

#### Steps to execute

###### Modify config in run_membus.sh
* region="me-south-1"
* number_of_victims=15
* number_of_adv=13
* account="diff/same"
* membus_fn="thrashers"
* uv=Victim URL
* ua=Adversary URL
* victim_size=128
* adv_size=3008

###### Command to execute-
* First argument - Run ID
* Second argument - Length of the interval

**sh run_membus.sh 1 2**
